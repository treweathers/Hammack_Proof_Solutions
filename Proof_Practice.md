### Practice Exam: Mathematical Proofs

**Instructions:** For each of the following statements, determine if it is true or false. If it is true, provide a formal proof. If it is false, provide a counterexample and a brief explanation (disproof).

1. Let $n \in \mathbb{Z}$. If $n^2 - 6n + 5$ is even, then $n$ is odd.

2. For all integers $a, b,$ and $c$, if $a$ divides $bc$, then $a$ divides $b$ or $a$ divides $c$.

3. There exist no integers $a$ and $b$ such that $21a + 35b = 1$.

4. For any three sets $A, B,$ and $C$, if $A \cup C = B \cup C$, then $A = B$.

5. For every positive integer $n$, 
   $$1^2 + 2^2 + 3^2 + \dots + n^2 = \frac{n(n+1)(2n+1)}{6}$$

6. For any sets $A$ and $B$, $\mathcal{P}(A) \cup \mathcal{P}(B) = \mathcal{P}(A \cup B)$.

7. If $n$ is an odd integer, then $8$ divides $n^2 - 1$.

8. For any integer $n \ge 0$, 
   $$\sum_{k=0}^{n} \binom{n}{k} = 2^n$$
