### Practice Exam: Mathematical Proof Techniques

**Instructions:** For each of the following statements, determine if it is true or false. 
If it is true, provide a formal proof. If it is false, provide a counterexample (disproof).
The specific method of proof is not provided; you must choose the most appropriate 
technique (Direct, Contrapositive, Contradiction, Induction, etc.).

---

#### 1. Divisibility and Parity (Chapter 4/5)
**Statement:** Let $n \in \mathbb{Z}$. If $n^2 - 6n + 5$ is even, then $n$ is odd.

#### 2. Rationality (Chapter 6)
**Statement:** The sum of an irrational number and a rational number is always irrational.

#### 3. Non-Conditional Statements (Chapter 7)
**Statement:** There exists no smallest positive real number.

#### 4. Set Theory (Chapter 8)
**Statement:** For any sets $A, B,$ and $C$, if $A \cup C = B \cup C$, then $A = B$.

#### 5. Disproof (Chapter 9)
**Statement:** For every prime number $n$, $2^n - 1$ is prime.

#### 6. Mathematical Induction (Chapter 10)
**Statement:** For every $n \in \mathbb{N}$, $1^3 + 2^3 + 3^3 + \dots + n^3 = \left(\frac{n(n+1)}{2}\right)^2$.

#### 7. Contrapositive/Contradiction (Chapter 5/6)
**Statement:** Let $x, y \in \mathbb{R}$. If $x^2 + y^2 = 0$, then $x = 0$.

#### 8. Combinatorial Proof (Chapter 3.10)
**Statement:** Prove the following identity using a counting argument (Double Counting):
$n 2^{n-1} = \sum_{k=1}^{n} k \binom{n}{k}$

#### 9. Sets and Power Sets (Chapter 8)
**Statement:** For any sets $A$ and $B$, $\mathcal{P}(A) \cup \mathcal{P}(B) = \mathcal{P}(A \cup B)$.

#### 10. Induction and Divisibility (Chapter 10)
**Statement:** For every $n \in \mathbb{N}$, $3$ divides $n^3 + 2n$.

#### 11. Direct Proof (Chapter 4)
**Statement:** If $n$ is an odd integer, then $8$ divides $n^2 - 1$.

#### 12. Existence and Uniqueness (Chapter 7)
**Statement:** There exists a unique set $A$ such that for every set $B$, $A \cup B = B$.
